# 경제 시뮬레이션
국가의 경제 정책(재난 지원금, 민생회복 소비쿠폰 등)이 국가 경제에 어떤 영향을 미칠 수 있는지(인플레이션, 금리 등) 시뮬레이션 해보는 프로젝트입니다.

## 기획 배경

- 2025년 발생한 물가 상승이 당해 년도에 배급된 소비 쿠폰때문에 발생한 현상인지, 그리고 소비 쿠폰으로 얻은 긍정적인 효과(GDP 상승, 국민 만족도 등)는 어느정도인지 알아보기 위해 기획된 프로젝트입니다.

## 구조

backend: python 3.14
frontend: react, TypeScript

## 특징

- 백엔드를 python으로 구축해 간단히 실행할 수 있습니다.

## 실행
```
npm run dev
```
